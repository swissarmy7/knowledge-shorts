# Shorts Script Prompt Engine

## 콘텐츠 포맷
- **발표자 없음**: 순수 내레이터 중심 비주얼 스토리텔링 (발표자 캐릭터·칠판 구도 없음)
- **목소리 톤**: 신뢰감 있는 중저음 내레이터 (구어체, 구체적, 리듬감)

## 스토리 아크 (바이럴 5단계)
1. **HOOK (장면 1-2)**: 손해·반전·숫자로 첫 3초 안에 스크롤 정지
2. **TENSION BUILD (장면 3-4)**: 공감대 형성 + 왜 중요한지 긴장감 고조
3. **CORE FACTS (중간)**: 한 장면 = 한 포인트, 구체적 수치·비교로 신뢰 확보
4. **REVEAL / PAYOFF (뒷부분)**: 앞에서 쌓인 긴장의 해소, 핵심 반전 공개
5. **OUTRO (마지막)**: 1줄 요약 + 논쟁 유도 댓글 CTA

## 타이밍 기준 (script_generator.py와 동기화)
| 장면 수 | 목표 길이 | 발화량 목표 |
|--------|---------|-----------|
| 8장면  | 60초    | ~318자     |
| 10장면 | 88초    | ~466자     |
| 12장면 | 108초   | ~572자     |

## 제목 구조 (두 줄 카피)
- **highlight** (노란색): 공백 제외 정확히 10자. 주제+상황+손해/반전 압축.
- **rest** (흰색): 공백 제외 정확히 12자. 왜 봐야 하는지 이유/결과/경고.
- 한 문장을 반토막 내지 말 것. 각각 독립된 카피여야 함.

## 팩트 안전 원칙
- 참고 자료(Google Search)에서 확인된 사실만 사용
- 불확실한 수치·규정은 "일반적으로", "약 OO", "알려진 바로는" 처리
- 카드/금융/의료/법률 정보는 확인된 사용 팁 위주

## 이미지 프롬프트 원칙
- `blackboard_content`: 영어 전용, 글자/숫자/기호 없는 순수 비주얼 묘사
- Full-frame 메인 비주얼. 발표자·강사·칠판 구도 금지.
